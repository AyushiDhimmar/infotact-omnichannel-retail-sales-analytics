# Omnichannel Retail Sales Analytics

## Overview

This project analyzes retail sales data to generate business insights using data cleaning, SQL queries, and dashboard visualization.

---

## Tech Stack

- Python (Data Cleaning)
- MySQL (SQL Analysis)
- Tableau (Dashboard)

---

## Project Structure

- data/ → raw and cleaned datasets
- notebooks/ → data cleaning scripts
- sql/ → analysis queries
- docs/ → insights and data dictionary
- dashboard/ → Tableau files

---

## Key Insights

- Total Revenue: ~2.29M
- Avg Order Value: ~230
- Top Category: Technology
- Top City: New York
